Installation guide

1. Create new Angular project with Tailwind CSS

You may select the Tailwind CSS option when creating your project, or follow this guide to add Tailwind to an existing project.

Next, install portal-ui-ng:

npm i portal-ui-ng

2. Update style.css

@import "tailwindcss"; @source "../node_modules/portal-ui-ng"; @import 'portal-ui-ng/assets/base/_index.css'; @import 'portal-ui-ng/assets/components/_index.css'; @import 'portal-ui-ng/assets/pages/_index.css';

3. Update app.config.ts

import { provideAnimations } from '@angular/platform-browser/animations'; import { provideLocalStorage, providePlatformDetector, provideTheme } from 'portal-ui-ng'; export const appConfig: ApplicationConfig = {   providers: [     ... your existing providers     provideAnimations(),     provideTheme(),     providePlatformDetector(),     provideLocalStorage(),   ] };

4. Add Material Symbols Rounded font

Add the following line to the HEAD of your index.html file:

<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Material+Symbols+Rounded:opsz,wght,FILL,GRAD@20..48,100..700,0..1,-50..200"/>
